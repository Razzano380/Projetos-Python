# Engenharia de Dados & Automação de Processos 🐍⚙️

Bem-vindo ao meu portfólio de dados! Sou **Juan Luccas G. Razzano**, Engenheiro com sólida base em otimização de processos.. Este repositório é dedicado à aplicação de Python e SQL para transformar fluxos manuais em sistemas automatizados e inteligentes.

## 🛠️ Stack Tecnológica

* **Linguagem Principal**: Python 3
* **Arquitetura de Software**: Desenvolvimento Orientado a Objetos (OOP) para sistemas modulares e escaláveis
* **Manipulação & ETL**: Domínio da biblioteca Pandas para tratamento de bases de dados complexas (Excel/CSV)
* **Análise de Dados**: Gestão de séries temporais com bibliotecas de Datetime e análise estatística básica.
* **Coleta de Dados**: Implementação de Web Scraping para captura de dados estruturados na web ;
* **Banco de Dados**: SQL avançado (Window Functions, Joins complexos, Transactions e Agregações)
* **Business Intelligence**: Desenvolvimento de dashboards estratégicos em Power BI

## 🚀 Projetos e Soluções

### 1. Automação de Indicadores Corporativos (Excel to Insights)
Desenvolvimento de motores em Python que automatizam a leitura de bases de dados industriais e financeiras.
* **Impacto**: Eliminação de tarefas repetitivas de consolidação de dados.
* **Tecnologia**: Python, Pandas e lógica de ETL.

### 2. Arquiteturas Escaláveis com OOP
Refatoração de scripts procedurais para modelos baseados em classes e objetos
* **Foco**: Reutilização de código, facilidade de manutenção e estabilidade de sistemas em produção.

### 3. Extração Automática (Web Scraping)
Scripts para coleta de dados externos, permitindo monitoramento de mercado e integração de fontes externas ao modelo de dados local
